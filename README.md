# Global YouTube Trending Videos Analysis

This project performs Exploratory Data Analysis (EDA) on YouTube trending videos dataset to identify patterns, trends, and engagement metrics.

## 📊 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Analysis
- Analyzed views, likes, comments, and engagement patterns
- Identified trends across different countries and categories
- Performed data cleaning and transformation

## 📈 Outcome
- Generated insights on video performance and audience engagement
- Visualized trends using graphs and charts
